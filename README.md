# Datankasittely ja koneoppiminen

### Tässä repositoriossa on Metropolia ammattikorkeakoulun otsikossa mainitulla kurssilla tekemiäni ryhmätyötehtäviä ja projekti. Pidin kurssista todella paljon ja se sujui hyvin ja opin paljon uusia asioita. Olin ennen IT-alan opintoja työssä K-kaupoissa ja voisin esimerkiksi soveltaa oppimaani jo ennestään tuttuun kaupan kassajärjestelmään vaikkapa saadakseni dataa kävijöiden ostoksista ja ennakoida näin uusien hankintojen tarvetta.
